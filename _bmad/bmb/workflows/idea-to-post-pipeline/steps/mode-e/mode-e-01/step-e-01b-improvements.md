# EDIT E-01b: Analyze and Recommend Improvements

## STEP GOAL:

Analyze [N] selected posts and recommend specific improvements for each (hook, problem, solution, CTA).

## EXECUTION:

**ALWAYS halt and wait for user input after presenting menu or asking for selection.**
**ONLY proceed to next step when user provides explicit input/selection.**

### 2. Analyze Each Post

For each selected post, analyze:

**POST #145 "Как начать с нуля"**
```
Анализирую пост #145...

📊 ТЕКУЩИЕ МЕТРИКИ:
  • CTR: 1.2%
  • Engagement: 3.4%
  • Quality Score: 68/100
  • Дата публикации: 20 дец

🔴 ПРОБЛЕМЫ НАЙДЕНЫ (3):
  ❌ Слабый хук — "Как начать" звучит размыто
     Предложение: "Начни с нуля и заработай за месяц"

  ❌ Отсутствует проблема пользователя в начале
     Предложение: Добавить "Не знаешь с чего начать?"

  ❌ CTA слабый (только ссылка)
     Предложение: "Скачай чек-лист и начни сегодня"

✅ РЕКОМЕНДУЕМЫЕ УЛУЧШЕНИЯ:

  ВАРИАНТ 1 (Острый хук + эмоция):
  "Начни с нуля за месяц (пока конкуренты спят) —
   7 шагов которых они боятся"

  ВАРИАНТ 2 (Интрига + обещание):
  "Не знаешь с чего начать? Вот что сделал я в первый месяц
   (и заработал свои первые $XXX)"

  ВАРИАНТ 3 (Статистика + решение):
  "73% начинающих бросают за первый месяц. Вот как не стать
   одним из них (и выйти в плюс за 30 дней)"

💚 ПРЕИМУЩЕСТВА ЭТИХ ВАРИАНТОВ:
  ✓ CTR должен вырасти на ~40-60% (до 1.8-2.0%)
  ✓ Более целевая аудитория
  ✓ Сильнее эмоциональный отклик
```
