# Solution Selection Templates

## Confidence Assessment Templates

### Overall Confidence Calculation Template
```
Overall_Confidence = (Evidence_Strength × Analytical_Rigor × Perspective_Balance) ^ Adjustment_Factor

Where:
- Evidence_Strength: Average of all source credibility and recency scores
- Analytical_Rigor: Assessment of methodological soundness and bias mitigation
- Perspective_Balance: Harmonic mean of technical, business, operational, risk scores
- Adjustment_Factor: 0.8-1.2 based on stakeholder alignment and contextual factors
```

### Confidence Interpretation Guide
- **0.9+**: Exceptional confidence - implement immediately
- **0.8-0.9**: High confidence - standard risk mitigation
- **0.7-0.8**: Good confidence - caution and monitoring
- **0.6-0.7**: Moderate confidence - pilot implementation
- **0.5-0.6**: Low confidence - further analysis required
- **<0.5**: Very low confidence - reassess decision framework

## Risk-Adjusted Value Templates

### Risk Components Assessment
- **Implementation Risk**: Probability × cost of failure
- **Operational Risk**: Business continuity impact × likelihood
- **Opportunity Risk**: Missed benefits × competitive disadvantage

### Risk-Adjusted Value Formula
```
Risk_Adjusted_Value = Expected_Value × (1 - Risk_Impact) × Confidence_Level
```

## Recommendation Confidence Levels

### Implementation Guidance by Confidence

#### Immediate Implementation (Confidence >0.9)
- **Action**: Proceed with standard project governance
- **Monitoring**: Regular milestone reviews
- **Contingency**: Basic risk mitigation only

#### Standard Implementation (Confidence 0.8-0.9)
- **Action**: Implement with enhanced monitoring
- **Risk Mitigation**: Address identified risk factors
- **Contingency**: Developed backup plans

#### Cautious Implementation (Confidence 0.7-0.8)
- **Action**: Implement as pilot with evaluation gates
- **Risk Mitigation**: Comprehensive risk management
- **Contingency**: Multiple fallback options

#### Pilot Implementation (Confidence 0.6-0.7)
- **Action**: Pilot program with full evaluation
- **Risk Mitigation**: Extensive controls and monitoring
- **Contingency**: Easy rollback capability

#### Further Analysis Required (Confidence 0.5-0.6)
- **Action**: Additional research and validation
- **Next Steps**: [specific_analysis_requirements]
- **Timeline**: [additional_time_needed]

#### Framework Reassessment (<0.5)
- **Action**: Reassess entire decision framework
- **Root Cause**: [confidence_limitations_identified]
- **Recommendations**: [framework_improvements_needed]
