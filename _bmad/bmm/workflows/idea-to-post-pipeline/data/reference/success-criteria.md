- ‚úÖ STRONG: Curiosity/Urgency/Pattern-Interrupt (score: 90%+)
- ‚ö†Ô∏è  MEDIUM: Some hook element present (score: 75-89%)
- ‚ùå WEAK: Generic or no hook (score: <75%)

- ‚úÖ CLEAR: Specific pain point for target audience (score: 90%+)

- ‚úÖ RELEVANT: Directly solves the stated problem (score: 90%+)

## Failure Indicators (What to Avoid)

### Quality Failure
- ‚ùå More than 20% of posts scoring <70%
- ‚ùå Consistency issues (hooks, length, tone vary wildly)
- ‚ùå Grammar/typo errors in >10% of posts
- ‚ùå No improvement despite feedback

**Action:** Review post-quality-checklist.md and improve process

### Production Failure
- ‚ùå Producing <3 posts per week (too slow)
- ‚ùå CTR trending downward over 3+ weeks
- ‚ùå Same topics repetitive (audience bored)
- ‚ùå Missing deadlines for content

**Action:** Switch modes, try YOLO for faster output, or analyze what's wrong

### Engagement Failure
- ‚ùå <1% engagement rate (audience not responding)
- ‚ùå Comments are negative or confused
- ‚ùå No shares (posts not shareable enough)
- ‚ùå Audience questions same things repeatedly

**Action:** Improve hook/CTA clarity, add more emotional triggers

### System Failure
- ‚ùå Workflow state not saving (can't resume)
- ‚ùå CSV files getting corrupted
- ‚ùå Metrics not tracking correctly
- ‚ùå Process breaking at certain steps

**Action:** Review data management, backup files, check file paths
