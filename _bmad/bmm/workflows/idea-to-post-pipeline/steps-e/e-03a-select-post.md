# EDIT E-03a: Select Post for A/B Testing

## STEP GOAL:

Select a single post for generating and comparing multiple variants.

## EXECUTION:
### EXECUTION RULES - HALT AND WAIT

**ALWAYS halt and wait for user input after presenting menu or asking for selection.**
**ONLY proceed to next step when user provides explicit input/selection.**


### 1. Display Selection Header

```
═══════════════════════════════════════════════════════════════

  🧪 A/B TEST: Генерирование вариантов для тестирования

  Давай найдём пост который хочешь протестировать.
  Я генерирую 3 варианта с разными углами.

═══════════════════════════════════════════════════════════════
```

### 2. Present Selection Options

```
ВЫБЕРИ ЧТО ТЕСТИРОВАТЬ:

[1] 📍 RECENT — Последние посты (последние 7 дней)
    Вот что опубликовано недавно

[2] 🔴 UNDERPERFORMING — Посты с низким CTR
    CTR < 2% — хорошие кандидаты для улучшения

[3] 🎯 BY TOPIC — По теме/категории
    Выбери категорию, покажу посты

[4] 🔍 SEARCH — Поиск по тексту
    Вбей часть названия или идею

[5] 💚 BEST PERFORMING — Топ посты
    Посты с CTR > 4% (можем сделать ещё лучше)

[6] 🎲 RANDOM — Случайный пост
    Выберу произвольно из базы

[L] LIST ALL — Показать все посты

═══════════════════════════════════════════════════════════════
```

Wait for user selection.
