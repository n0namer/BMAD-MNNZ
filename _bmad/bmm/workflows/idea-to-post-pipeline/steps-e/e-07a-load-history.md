# EDIT E-07a: Load Version History

## STEP GOAL:

Load and display version history for a selected post.

## EXECUTION:
### EXECUTION RULES - HALT AND WAIT

**ALWAYS halt and wait for user input after presenting menu or asking for selection.**
**ONLY proceed to next step when user provides explicit input/selection.**


### 1. Display Selection Header

```
═══════════════════════════════════════════════════════════════

  📜 ИСТОРИЯ ВЕРСИЙ: Эволюция поста

  Давай посмотрим как менялся твой пост

═══════════════════════════════════════════════════════════════
```

### 2. Present Selection Options

```
ВЫБЕРИ ПОСТ ДЛЯ ПРОСМОТРА ИСТОРИИ:

[1] 🔴 RECENT — Свежие посты с историей
    Посты которые недавно обновлялись

[2] 🟠 EDITED — Посты которые редактировали много раз
    Посты где есть v1, v2, v3...

[3] 🟢 STABLE — Посты без изменений
    Оригинальные посты которые не трогали

[4] 🔍 SEARCH — Поиск по названию
    Введи название и найду историю

[5] 💚 TOP POSTS — Топ посты с историей
    Самые успешные посты со всеми версиями

[L] LIST ALL — Показать все посты

═══════════════════════════════════════════════════════════════
```

Wait for user selection.

### 3. Display Posts for Selection

**[1] RECENT:**
```
📅 СВЕЖИЕ ПОСТЫ С ИСТОРИЕЙ (последние 7 дней, у которых есть версии):

[1] #156 "Новое об AI" (3 дня) | Версии: v1 → v2 (2 версии)
[2] #154 "Инсайт дня" (4 дня) | Версии: v1 → v2 → v3 (3 версии)
[3] #152 "Как я заработал" (5 дней) | Версии: v1 (1 версия, не менялся)
[4] #150 "Ошибка" (6 дней) | Версии: v1 → v2 (2 версии)
[5] #148 "5 инструментов" (7 дней) | Версии: v1 → v2 → v3 (3 версии)

Выбери номер для просмотра истории (1-5): ___
```

**[2] EDITED:**
```
🟠 ПОСТЫ С МНОЖЕСТВЕННЫМИ РЕДАКЦИЯМИ (>2 версии):

[1] #154 "Инсайт дня" | Версии: 3 (v1, v2, v3)
    Последний апдейт: 4 дня назад
    История редакций: Hook улучшен → CTA усилен

[2] #148 "5 инструментов" | Версии: 3 (v1, v2, v3)
    Последний апдейт: 7 дней назад
    История редакций: Примеры добавлены → Структура улучшена

[3] #145 "Как начать с нуля" | Версии: 2 (v1 → v2)
    Последний апдейт: Вчера
    История редакций: Hook переписан полностью

Выбери номер (1-3): ___
```

**[4] SEARCH:**
```
🔍 ПОИСК ПО НАЗВАНИЮ

Введи часть названия поста: ___

Найдено 3 поста:
[1] #145 "Как начать с нуля" | v1 → v2
[2] #140 "Как создать курс" | v1
[3] #152 "Как я заработал" | v1

Выбери номер: ___
```

### 4. Confirm Selection

```
═══════════════════════════════════════════════════════════════

✅ ВЫБРАН ПОСТ:

#154 "Инсайт дня"

Версии в истории: 3
  • v1: Оригинал (4 дня назад)
  • v2: Улучшение (3 дня назад)
  • v3: Последний (1 день назад)
```
