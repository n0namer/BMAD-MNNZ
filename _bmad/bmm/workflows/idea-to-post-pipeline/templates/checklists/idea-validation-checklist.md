# Идея-Валидация Чек-лист

## Критерии валидации идеи перед исследованием

### 1. Специфичность идеи ✓

- [ ] **Идея конкретная, не общая**
  - ✅ GOOD: "ИИ сокращает время написания контента с 2 часов до 20 минут"
  - ❌ BAD: "ИИ помогает"

- [ ] **Есть понятный элемент value proposition**
  - ✅ GOOD: "Как использовать ИИ для экономии времени"
  - ❌ BAD: "Информация об ИИ"

### 2. Исследуемость идеи ✓

- [ ] **Есть достаточно контента в интернете по теме**
  - Минимум: 5-10 источников доступны
  - Максимум: тема не слишком узкая

- [ ] **Идея не слишком узко специальная**
  - ✅ GOOD: "ИИ и контент" (широкая тема, много углов)
  - ❌ BAD: "Специфическая конфигурация GPT-4 для моего случая"

- [ ] **Можно найти минимум 5-8 разных углов**
  - Для исследования нужна глубина, не ширина

### 3. Аудитория понятна ✓

- [ ] **Ясна целевая аудитория**
  - Для кого эта идея? (фаундеры, маркетеры, копирайтеры, и т.д.)
  - Есть ли у них проблема, которую решает идея?

- [ ] **Аудитория достаточно большая**
  - ✅ GOOD: "Для фаундеров и маркетеров" ( 1000s потенциальной аудитории)
  - ❌ BAD: "Для людей использующих GPT-4 API в конфигурации X" (10s)

### 4. Не дубликат ✓

- [ ] **Идея не была недавно использована**
  - Не опубликована в последние 3 месяца
  - Или есть новый угол на старую тему

- [ ] **Отличается от других идей в очереди**
  - Проверить ideas_inbox перед добавлением новой

### 5. Интерес/Вовлечённость ✓

- [ ] **Идея вызывает интерес**
  - Есть ли в идее элемент curiosity, urgency, или social proof?
  - Будет ли аудитория реагировать (комментарии, репосты)?

- [ ] **Есть потенциал для диалога**
  - Можно ли задать вопрос аудитории?
  - Будут ли люди обсуждать?

---

## Оценка: PASS / NEEDS WORK / REJECT

### PASS ✅
Все критерии выполнены:
- Идея конкретная и понятная
- Есть достаточно материала для исследования
- Аудитория ясна и достаточно большая
- Есть потенциал для вовлечения

→ Переходим к исследованию

### NEEDS WORK ⚠️
1-2 критерия не полностью выполнены:
- Уточнить или переформулировать идею
- Найти дополнительные источники
- Сузить или расширить аудиторию

→ Доработать идею и валидировать снова

### REJECT ❌
3+ критерия не выполнены:
- Идея слишком расплывчатая
- Слишком мало материала для исследования
- Аудитория непонятна или слишком узкая
- Слишком похоже на другой пост

→ Отклонить идею, вернуться к ideas_inbox для выбора другой
